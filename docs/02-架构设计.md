## 3. 核心模块设计

### 3.1 数据模型

#### 3.1.1 TreeNode（通用树节点）
```typescript
interface TreeNode<T = any> {
  id: string;
  parentId: string | null;
  children: string[];
  data: T;
  metadata: {
    createdAt: number;
    updatedAt: number;
    collapsed: boolean;
  };
}
```

#### 3.1.2 ChatNode（对话节点）
```typescript
interface ChatNode extends TreeNode<ChatNodeData> {
  data: {
    role: 'user' | 'assistant' | 'system';
    content: string;
    status: 'pending' | 'streaming' | 'completed' | 'aborted';
    branchLabel?: string;
    toolCalls?: ToolCall[];
  };
}
```

#### 3.1.3 WorkTreeNode（工作树节点）
```typescript
interface WorkTreeNode extends TreeNode<WorkTreeData> {
  data: {
    type: 'branch' | 'commit' | 'file';
    name: string;
    path: string;
    status: 'active' | 'stale' | 'merged';
    changes?: FileChange[];
  };
}
```

### 3.2 状态管理（Zustand）

```typescript
// chatTreeStore.ts
interface ChatTreeStore {
  // 状态
  nodes: Map<string, ChatNode>;
  rootNodes: string[];
  focusedNodeId: string | null;

  // 操作
  addNode: (node: ChatNode) => void;
  updateNode: (id: string, updates: Partial<ChatNode>) => void;
  deleteNode: (id: string) => void;
  setFocus: (id: string) => void;

  // 树操作
  getNodePath: (id: string) => ChatNode[];
  getChildren: (id: string) => ChatNode[];
  getDescendants: (id: string) => ChatNode[];
}
```

### 3.3 树形布局引擎

```typescript
// TreeLayout.ts
interface TreeLayoutOptions {
  direction: 'vertical' | 'horizontal' | 'radial';
  nodeSpacing: number;
  levelSpacing: number;
  animate: boolean;
}

class TreeLayout {
  constructor(options: TreeLayoutOptions);

  // 计算节点位置
  layout(nodes: TreeNode[]): LayoutResult;

  // 增量更新
  updateLayout(changedNodes: TreeNode[]): LayoutResult;
}

interface LayoutResult {
  positions: Map<string, Position>;
  edges: Edge[];
  bounds: Bounds;
}
```

## 5. 数据流设计

### 5.1 创建分支流程

```
用户点击"创建分支"
    ↓
BranchInput 组件收集输入
    ↓
调用 chatTreeStore.addNode()
    ↓
通过 WebSocket 发送到后端
    ↓
后端处理并返回新节点
    ↓
WebSocket 推送状态更新
    ↓
Store 更新节点状态
    ↓
组件重新渲染
```

### 5.2 实时状态同步

```typescript
// useWebSocket.ts
export function useWebSocket(url: string) {
  const updateNode = useChatTreeStore(state => state.updateNode);

  useEffect(() => {
    const ws = new WebSocket(url);

    ws.onmessage = (event) => {
      const message = JSON.parse(event.data);

      switch (message.type) {
        case 'node.update':
          updateNode(message.nodeId, message.updates);
          break;
        case 'node.streaming':
          // 处理流式更新
          break;
      }
    };

    return () => ws.close();
  }, [url]);
}
```

## 6. 性能优化策略

### 6.1 虚拟化渲染
- 只渲染可视区域内的节点
- 使用 `react-window` 或自定义虚拟滚动

### 6.2 增量布局
- 只重新计算变化的节点位置
- 缓存布局结果

### 6.3 节点折叠
- 折叠的子树不渲染
- 延迟加载子节点

### 6.4 防抖/节流
- 缩放、平移操作使用节流
- 搜索输入使用防抖
